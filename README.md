# ai-agent-frontend
A clean HTML/CSS/JavaScript frontend for interacting with a custom AI-Agent


## Features

- Chat Interface: Send messages to the AI agent and receive responses.
- Sidebar: Displays recent conversation history and a button for new queries.
- Dark/Light Mode: Toggle between dark and light themes using the switch.
- Keyboard Support: Press Enter or Tab to send messages.
- Responsive UI: Styled with CSS variables for a modern look.

## How to Use

1. Open `index.html` in your browser.
2. Type your message in the input field.
3. Press Enter, Tab, or click the send button to send your message.
4. Use the dark/light mode switch in the sidebar to change themes.
